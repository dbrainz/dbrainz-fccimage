Dbrainz's FCC Image Search Abstraction Layer

Usage : 'https://dbrainz-image.herokuapp.com/search/search terms?offset=1'

Search terms are separated by '%20'. For example, to search for 'cats' and 'dogs' the URL would be 'https://dbrainz-image.herokuapp.com/search/cats%20dogs?offset=1'

Offset can be any value between 1 and 91. If you do not supply an offset value, it will default to 1.

Returns a JSON object containing the following fields for the first 10 images returned by the search:
page_url : URL for the page containing the image
snippet : description of the image
image_url : direct URL of the image

Using the path '/latest' will return a JSON object containing information about the last 10 searches.
